# Exercice - Session 1

## Le composant Hello

Créer un composant `Hello` qui accépte 2 props: `name: string` et `age: number`. Ce composant doit retourner le JSX suivant :

```
<p>Bonjour {name}, vous avez {age} ans</p>
```

## Les props facultatives

Faire en sorte, dans le composant `Hello`, que la props `age` soit facultative.

Si la props n'est pas présente dans le composant, alors le JSX suivant doit être affiché :

```
<p>Bonjour {name}, je ne connais pas votre age</p>
```

## Ajouter du style

Grace à styled component, créer un composant `RedParagraph` (pas besoin de créer un nouveau fichier) qui doit mettre le text en rouge.

Utiliser se composant dans le composant `Hello`.
